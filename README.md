### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of web technologies (optional, for development)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sachinn-p/Quiz_App.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Quiz_App
   ```

3. Open the `index.html` file in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```

   Or simply double-click the `index.html` file in your file manager.
